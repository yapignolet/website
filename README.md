# Yvonne-Anne Pignolet — Personal Website

Static personal academic website, hosted on GitHub Pages.

**Live site:** https://www.yvonneanne.pignolet.ch

## Structure

- `index.html` — Home / About
- `cv.html` — Curriculum Vitae
- `publications.html` — Publications
- `projects.html` — Projects and Teaching
- `style/` — CSS and assets
- `favicon.ico`

## Hosting

Hosted via [GitHub Pages](https://pages.github.com/) on the `main` branch (root `/`).
The custom domain is configured via the `CNAME` file.
